## Reglas practicas

- Si vienes desde el menu del producto, usa el mismo nombre en esta referencia.
- Si buscas contrato exacto, entra al spec del dominio y luego al endpoint.
- Si buscas contexto funcional, empieza por las guias del tab **Guía**.
