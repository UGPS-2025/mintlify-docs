## Endpoint base

Tu API disponible para pruebas y consumo está en:

- **Base URL**: `https://api.raul.ugps.io`
- **Health check**: `https://api.raul.ugps.io/api/health`
- **Esquema OpenAPI**: `https://api.raul.ugps.io/api/openapi.json`

La pestaña de **API** renderiza automáticamente los endpoints definidos en el
esquema OpenAPI.
